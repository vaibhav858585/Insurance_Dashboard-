# Insurance_Dashboard

# Insurance Dashboard Project

# Designed and developed an interactive Insurance Dashboard to analyze insurance business performance and customer engagement using data visualization tools. The dashboard provides a complete overview of active and inactive policies, total customer meetings, yearly meeting trends, policy status distribution, and policy type-wise analysis. It also highlights top open business opportunities and client connection insights to support better strategic decision-making.

# This project focuses on transforming raw insurance data into meaningful visual reports that help users monitor policy performance, identify growth opportunities, and improve customer relationship management. The dashboard was created with user-friendly visuals, filters, and KPI indicators to make data analysis faster and more efficient for business users and management teams.
